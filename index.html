Greg Knothe Project Portfolio

Some personal projects I have worked on in my free time for self use or others.
YGORec
A user-friendly tool that analyzes all public high-ranking tournament decks for the Yu-Gi-Oh card game, distilling nearly a million cards down into auto-generated visuals and simple statistics that anyone can understand to better prepare them for future event.
key methods: data collection (web scraping and API), data cleaning, statistical analysis

Hand Probability Calculator
An easy to use tool to calculate the odds of drawing specific combinations of objects from user-set groups via quick simulations. Allows for complex groupings that would otherwise not be reasonably calculable by math.
Methods: simulation, statistics

Twitch Chatbot with OBS Integration
A simple Twitch chatbot with all the bells and whistles one would expect. In addition to the basics, it also has Open Broadcasting Software (OBS) integration to allow for chat commands to display visually on the stream itself. Also working on a collectable system that will be tied to the stream, accessible via website.
key methods: data collection (web scraping), real-time API, photoshop
